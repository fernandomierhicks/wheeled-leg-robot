## Controller Details

### 1. Balance LQR (inner, 500 Hz)

**State vector change from Session 4:**

```
OLD (broken for drive):  x = [pitch_err, pitch_rate, wheel_pos, wheel_vel]   вҶҗ 4-state
NEW:                     x = [pitch вҲ’ Оё_ref, pitch_rate, wheel_vel_avg вҲ’ v_ref]  вҶҗ 3-state
```

Removing `wheel_pos` eliminates the restoring force that fights intentional motion.
Position hold in BALANCE mode is handled by the outer loop (Оё_ref derived from drift).

**LQR formulation:**
```
u = вҲ’K @ x
K = [K_pitch, K_pitch_rate, K_vel]   вҶҗ expected ~[-35, -7, -3]
Q = diag([Q_pitch, Q_pitch_rate, Q_vel])
R = scalar
```

**Gain Scheduling (Ascento-style):**

The 4-bar stroke (~62В°) changes effective pendulum length l_eff by ~50 mm.
Gains are precomputed at 3 operating points and interpolated online:

```python
K_table = {
    'retracted': compute_lqr_gain(q_hip=Q_RET),   # legs short
    'nominal':   compute_lqr_gain(q_hip=Q_NOM),   # default stance
    'extended':  compute_lqr_gain(q_hip=Q_EXT),   # legs long
}

# Online at 500 Hz:
alpha = (q_hip - Q_RET) / (Q_EXT - Q_RET)
K = (1 - alpha) * K_table['retracted'] + alpha * K_table['extended']
```

l_eff is computed via `physics.get_equilibrium_pitch(p, q_hip)` at each table entry.

---

### 2. Velocity PI (outer, 50 Hz)

Converts velocity command into a lean angle setpoint for the LQR:

```python
v_err = v_desired вҲ’ v_measured
Оё_ref = Kp_v * v_err + Ki_v * integral(v_err)
Оё_ref = clamp(Оё_ref, вҲ’0.26 rad, +0.26 rad)   # Вұ15В° hard limit
```

In BALANCE mode: `Оё_ref = K_pos * wheel_pos_drift` (gentle position hold), `v_ref = 0`.
In DRIVE mode: `Оё_ref` from PI above, `v_ref = v_desired` (feedforward to LQR).

Starting gains: Kp_v = 0.04 rad/(m/s), Ki_v = 0.008 rad/m.

---

### 3. Yaw PI (outer, 50 Hz)

Produces differential torque for turning. Blends with drive.

```python
yaw_err = Пү_yaw_desired вҲ’ Пү_yaw_measured   # from BNO086 gyro Z
П„_yaw = Kp_yaw * yaw_err + Ki_yaw * integral(yaw_err)
П„_yaw = clamp(П„_yaw, вҲ’0.5 NВ·m, +0.5 NВ·m)

П„_L = П„_sym + П„_yaw
П„_R = П„_sym вҲ’ П„_yaw
```

Starting gains: Kp_yaw = 0.3, Ki_yaw = 0.05.

---

### 4. Leg Impedance PD (independent, 100 Hz)

Decoupled from balance. Controls hip motor as a spring-damper:

```python
П„_hip = вҲ’K_s * (q_hip вҲ’ q_nom) вҲ’ B_s * dq_hip
```

| Setting | K_s (NВ·m/rad) | B_s (NВ·mВ·s/rad) |
|---------|--------------|-----------------|
| Soft    | 1.0          | 0.3             |
| Medium  | 5.0          | 0.8             |
| Stiff   | 15.0         | 1.5             |

`SUSPENSION_SOFTNESS` scalar 0вҶ’1 interpolates K_s and B_s.
`q_nom` = commanded leg height (adjustable by user).

**Terrain adaptation:** When unexpected pitch disturbance detected (|pitch вҲ’ Оё_ref| > 0.05 rad
while Оё_ref вүҲ 0), reduce K_s by 30% for 200ms to absorb impact passively.
